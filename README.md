# AI DDR Report Generator

This project converts inspection and thermal reports into a structured Detailed Diagnostic Report (DDR).

## Features
- Extracts text from PDF reports
- Analyzes thermal data
- Generates structured diagnostic report

## Approach
1. Extract data from inspection & thermal reports
2. Identify temperature variations
3. Generate client-friendly DDR report

## Tech Used
- Python
- PyMuPDF
- Streamlit / Google Colab

## Limitations
- Limited image-level analysis
- Depends on input data quality

## Future Improvements
- Full web UI
- Advanced AI models
- Image mapping integration
